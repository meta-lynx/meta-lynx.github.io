Data Loading

In this section we will discuss how to load the data prepared in Data Preparation into the Metascatter tool:

  1. Loading the CSV
  2. Configuring the CSV

Loading the CSV

To load a new data file click on Upload New CSV

Configuring the CSV

If loading a dataset for the first time, you will need to supply the configuration parameters via Manage Dataset (you should automatically be prompted to do this via the Assign fields button).

For each column of the CSV, you should select the data type (for instance, image path, scalars or categorical types). If the image paths are stored as a relative link, you will need to enter the prefix of the absolute path, e.g.:

In the case of the segmentation csv, Type should be Categorical and IoUs should be Scalar.

You also need to identify which columns correspond to the co-ordinates of the scatter plot. At least two columns need to be selected for each co-ordinate system. Change the Projection Name field away from any of the column names (for instance, you could choose one that represents how the co-ordinates were calculated e.g. unet_layer25).

You can include more that one set of co-ordinates, for instance, if you have feature embeddings from different models or ones extracted from different layers. These can be selected in the viewer using the Projection selector under the File Operations menu box.
